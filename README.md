# Uii 

*UI experiments based on the Nintendo Wii Theme!*

## How to run

* Clone this repository  
  `git clone git@github.com:fusetim/uii.git`
* Install the dependencies  
  `npm i`
* Run the dev server  
  `npm run dev`
